# 🦈 Wireshark: Privilege Management and ICMPv6 Analysis

## Objective
This lab demonstrates how to configure Wireshark for non-root packet capture using the principle of Least Privilege (Hardening), and how to perform basic protocol analysis by capturing and inspecting ICMPv6 traffic.

## Tools & Environment
- **OS:** Linux (Ubuntu)
- **Tools:** Wireshark, Dumpcap, Bash Terminal
- **Frameworks/Concepts:** Least Privilege, CIA Triad (Availability)

## 📁Directory Structure
- `shell.sh`: Commands used for privilege management and hardening.
- `analys.md`: Technical analysis of the captured network traffic.
- `readme.pt-br.md`: Portuguese version of this documentation.
