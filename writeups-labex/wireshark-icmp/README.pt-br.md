# 🦈 Wireshark: Gestão de Privilégios e Análise ICMPv6

## Objetivo
Este laboratório demonstra como configurar o Wireshark para captura de pacotes sem a necessidade de usuário root, aplicando o princípio do Privilégio Mínimo (Hardening), e como realizar uma análise básica de protocolos capturando e inspecionando tráfego ICMPv6.

## Ferramentas e Ambiente
- **SO:** Linux (Ubuntu)
- **Ferramentas:** Wireshark, Dumpcap, Terminal Bash
- **Conceitos:** Privilégio Mínimo, Triade CIA (Disponibilidade)

## Estrutura do Diretório
- `shell.sh`: Comandos utilizados para gestão de privilégios e hardening do binário.
- `analysy.pt-br.md`: Análise técnica do tráfego de rede capturado.
- `readme.md`: Versão em inglês desta documentação.
